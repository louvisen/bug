Bug:

In the software realm, a bug is an error, fault or a flaw in a software or system. Bugs are usually created due to human error and can produce an incorrect or unexpected result or cause a program to crash. These can generate issues with different levels of impact and severity. To help detect and avoid bugs, rigorous and thorough testing needs to be incorporated in the development process.
